# 販売実績分析の使い方

```text
1. 販売管理システムから CSV または Excel をエクスポートし、Sales/Data/ に置く
   （ファイル名は自由。列構成もそのままでよい）

2. Claude Code で実行：
   /analyze-sales 2026-06     ← 月次
   /analyze-sales             ← 全期間

3. 初回はデータ定義書 Sales/SCHEMA.md が生成される。
   ［要確認］の箇所（金額の単位・1行の意味など）を直しておくと以降の分析精度が上がる
```

- レポートは `Sales/Reports/SalesReport_期間.md` に生成される（グラフ付き）
- データを追加して同じ期間で再実行すれば、レポートは上書き更新される
- **機密データにつき、このリポジトリは必ずプライベートのまま運用すること**
